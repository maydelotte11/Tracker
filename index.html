Connect The Keto Ledger

This app syncs your data across every device through a free Firebase project (Google's backend service). You only need to do this setup once per browser.

  1. Go to console.firebase.google.com and create a free project.
  2. In the project, click Build → Firestore Database → Create database (start in production mode, any region).
  3. Click Build → Authentication → Get started, then enable the Email/Password sign-in method.
  4. Go to Project settings → General, scroll to "Your apps", click the </> (Web) icon to register a web app, and copy the firebaseConfig object it gives you.
  5. Paste that whole object below.
